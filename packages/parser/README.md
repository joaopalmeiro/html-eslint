# `@html-eslint/parser`

A parser for HTML-ESLint which parse HTML syntax.

## Installation

```bash
$ yarn add -D @html-eslint/parser
$ npm install -D @html-eslint/parser
```

## Usage

Set the `parser` property in your ESLint configuration file.

```json
{
  "parser": "@html-eslint/parser"
}
```
